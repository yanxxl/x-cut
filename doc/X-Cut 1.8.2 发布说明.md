## v1.8.2 带来字幕繁简转换、播放器修复和稳定性改进

### 新功能

- **字幕繁简转换** —— 新增字幕转码功能，支持将字幕在简体中文与繁体中文之间互相转换，方便不同地区用户使用。

### 问题修复

- **播放器文件切换修复** —— 修复重新选择文件后无法打开播放的问题，文件切换更加稳定可靠。
- **项目切换状态修复** —— 修复项目切换时状态重置不完整的问题，避免数据残留导致的异常行为。

## 功能总览

### 素材与字幕

| 功能                | 说明                                                               |
| ------------------- | ------------------------------------------------------------------ |
| 多格式导入          | 支持 MP4、WebM、MOV、MP3、WAV、JPG、PNG 等主流格式                 |
| 拖放导入            | 拖放文件或文件夹即可导入视频和字幕，并自动匹配字幕，操作更直观高效 |
| 导入外部字幕        | SRT、VTT、ASS/SSA、SUB、LRC、JSON 等格式，字幕文件可拖动排序       |
| 目录导入            | 选择目录，递归扫描子文件夹，自动按名称前缀匹配视频与字幕           |
| 生成空白字幕        | 按智能时长（约 5 分钟）自动分段，在自然停顿处切割                  |
| Whisper AI 语音识别 | 内嵌 Python + Whisper，默认 small 模型，自动生成带时间戳字幕       |
| 逐字时间稿          | 生成字幕同时输出 JSON 逐词时间戳，支持去除特定词等精确操作         |
| 智能场景分割        | 使用 scenedetect 自动检测场景切换点                                |
| 字幕文本编辑        | 双击直接编辑字幕文字内容                                           |
| AI 转写刷新         | 悬停字幕按 U 键调用 Whisper 重新识别该段文字                       |
| 去除特定词          | 基于逐字时间稿智能去除口语赘词等特定词汇                           |
| 字幕繁简转换        | 支持字幕在简体中文与繁体中文之间互相转换                           |
| 字幕文件复制        | 右键快速复制已有字幕                                               |
| 字幕导出文本        | 导出为纯文本文件，方便本地批量编辑                                 |
| VTT 字幕导出        | 将编辑后的字幕导出为 VTT 文件                                      |

### 字幕编辑

| 操作         | 说明                                                 |
| ------------ | ---------------------------------------------------- |
| 鼠标悬停     | 实时预览对应视频帧和音频                             |
| 点击         | 选中/取消选中字幕条目                                |
| 按住拖动     | 批量选中字幕                                         |
| Shift + 拖动 | 批量取消选中                                         |
| 反向选择     | 快速选中当前未选中的条目                             |
| B 键         | 分割字幕（基于时间比例）                             |
| D 键         | 合并分隔线两侧的字幕                                 |
| 空格键       | 播放/暂停当前视频                                    |
| P 键         | 截取当前帧到桌面                                     |
| I 键         | 编辑当前字幕条目                                     |
| U 键         | AI 重新识别当前字幕文字                              |
| S 键         | 切割当前段落（合并段落在段落标题栏扩展菜单）         |
| 段落点击     | 跳转到该段落或从该位置开始播放                       |
| 页面搜索     | 搜索字幕文字，支持 `?` `*` 模糊匹配，高亮 + 导航     |

### 时间线与导出

| 功能          | 说明                                               |
| ------------- | -------------------------------------------------- |
| 多时间线      | 创建多条独立时间线，互不干扰，支持拖拽排序         |
| 时间线复制    | 快速复制已有时间线                                 |
| 删除模式      | 点击删除不需要的字幕片段，配合导出实现精细剪辑     |
| 文稿匹配      | 粘贴文稿自动匹配字幕并生成时间线（专业版）         |
| 视频导出      | H.264 编码，CRF 18 高质量，支持自定义输出目录      |
| 合并相邻条目  | 导出时可选合并时间相邻的片段，提升导出效率         |
| 视频分割      | 长视频按 1 小时或 30 分钟分割                      |
| 音频导出      | 导出原声音频或压缩音频（适合 whisper 使用）        |
| FCPXMLD 导出  | 与 Final Cut Pro，达芬奇，剪映等无缝衔接（专业版） |
| FCP7 XML 导出 | 与 Adobe Premiere Pro 无缝衔接（专业版）           |
