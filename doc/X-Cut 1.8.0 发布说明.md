## v1.8.0 带来时间线删除模式、播放体验优化和多项改进

### 新功能

- **时间线删除模式** —— 新增删除模式，点击时间线上的字幕条目即可删除，方便快速清理不需要的片段；支持在删除模式下切换播放进度，实时预览效果。
- **字幕反向选择** —— 字幕段落支持反向选择功能，快速选中当前未选中的条目，批量操作更高效。
- **导出支持删除模式** —— 导出视频和 FCPXML/FCP7 XML 时，支持应用删除模式，只导出保留下来的片段。

### 体验优化

- **导出取消功能** —— 新增导出取消按钮，支持随时停止正在进行的导出任务。
- **导出性能优化** —— 自动合并相邻片段，优化编码过程，显著提升导出性能。
- **时间线播放重构** —— 重新设计时间线播放实现，以字幕为单位展示，播放逻辑更清晰完整。
- **播放流畅度提升** —— 优化时间线播放性能，相邻片段切换更流畅；解决高负载下的播放卡顿问题。
- **时间线性能优化** —— 当字幕条目过多时自动分组（每100条），提升渲染性能，同时保持播放进度可见。
- **播放控制改善** —— 解决暂停播放时时间线仍继续播放的问题，播放控制更精准。
- **当前字幕标识** —— 当前播放的字幕条目添加下划线标识，视觉反馈更清晰。
- **模式切换按钮优化** —— 改善时间线模式切换按钮样式，操作更直观。
- **波形图组件重构** —— 分离职责，代码结构更清晰，维护更方便。

### 问题修复

- 修复时间线数据偶发丢失的问题
- 修复时间线播放中多文件切换的异常问题
- 修复片段播放结束后暂停状态不一致的问题

## 功能总览

### 素材与字幕

| 功能                | 说明                                                               |
| ------------------- | ------------------------------------------------------------------ |
| 多格式导入          | 支持 MP4、WebM、MOV、MP3、WAV、JPG、PNG 等主流格式                 |
| 拖放导入            | 拖放文件或文件夹即可导入视频和字幕，并自动匹配字幕，操作更直观高效 |
| 导入外部字幕        | SRT、VTT、ASS/SSA、SUB、LRC、JSON 等格式，字幕文件可拖动排序       |
| 目录导入            | 选择目录，递归扫描子文件夹，自动按名称前缀匹配视频与字幕           |
| 生成空白字幕        | 按智能时长（约 5 分钟）自动分段，在自然停顿处切割                  |
| Whisper AI 语音识别 | 内嵌 Python + Whisper，默认 small 模型，自动生成带时间戳字幕       |
| 逐字时间稿          | 生成字幕同时输出 JSON 逐词时间戳，支持去除特定词等精确操作         |
| 智能场景分割        | 使用 scenedetect 自动检测场景切换点                                |
| 字幕文本编辑        | 双击直接编辑字幕文字内容                                           |
| AI 转写刷新         | 悬停字幕按 U 键调用 Whisper 重新识别该段文字                       |
| 去除特定词          | 基于逐字时间稿智能去除口语赘词等特定词汇                           |
| 字幕文件复制        | 右键快速复制已有字幕                                               |
| 字幕导出文本        | 导出为纯文本文件，方便本地批量编辑                                 |
| VTT 字幕导出        | 将编辑后的字幕导出为 VTT 文件                                      |

### 字幕编辑

| 操作         | 说明                                                 |
| ------------ | ---------------------------------------------------- |
| 鼠标悬停     | 实时预览对应视频帧和音频                             |
| 点击         | 选中/取消选中字幕条目                                |
| 按住拖动     | 批量选中字幕                                         |
| Shift + 拖动 | 批量取消选中                                         |
| 反向选择     | 快速选中当前未选中的条目                             |
| B 键         | 分割字幕（基于时间比例）                             |
| D 键         | 合并分隔线两侧的字幕                                 |
| 空格键       | 播放/暂停当前视频                                    |
| P 键         | 截取当前帧到桌面                                     |
| I 键         | 编辑当前字幕条目                                     |
| U 键         | AI 重新识别当前字幕文字                              |
| S 键         | 切割当前段落（合并段落在段落标题栏扩展菜单）         |
| 段落点击     | 跳转到该段落或从该位置开始播放                       |
| 页面搜索     | 搜索字幕文字，支持 `?` `*` 模糊匹配，高亮 + 导航 |

### 时间线与导出

| 功能          | 说明                                               |
| ------------- | -------------------------------------------------- |
| 多时间线      | 创建多条独立时间线，互不干扰，支持拖拽排序         |
| 时间线复制    | 快速复制已有时间线                                 |
| 删除模式      | 点击删除不需要的字幕片段，配合导出实现精细剪辑     |
| 文稿匹配      | 粘贴文稿自动匹配字幕并生成时间线（专业版）         |
| 视频导出      | H.264 编码，CRF 18 高质量，支持自定义输出目录      |
| 视频分割      | 长视频按 1 小时或 30 分钟分割                      |
| 音频导出      | 导出原声音频或压缩音频（适合 whisper 使用）        |
| FCPXMLD 导出  | 与 Final Cut Pro，达芬奇，剪映等无缝衔接（专业版） |
| FCP7 XML 导出 | 与 Adobe Premiere Pro 无缝衔接（专业版）           |
