## v1.4.0 带来多项重要新功能和体验优化

### 新功能

- **FCP7 XML 导出** —— 时间线右键菜单新增"导出 FCP7 XML（适用 Premiere）"，生成符合 FCP7 xmeml 5.0 规范的 XML 文件，可直接导入 Adobe Premiere Pro。
- **目录导入** —— 首页新增目录导入按钮，选择包含视频和字幕文件的目录，系统自动按名称前缀匹配字幕、创建项目并批量导入，一步到位。
- **字幕统计** —— 字幕标题右侧显示条目总数和字数统计（如 "128 条 · 3560 字"），方便快速了解字幕规模。
- **搜索模糊匹配** —— 页内搜索支持 `?`（匹配任意单字符）和 `*`（匹配任意多字符）通配符，搜索更灵活。
- **智能分段** —— 字幕段落分割算法升级，默认按约 5 分钟分段，并在分割点附近自动寻找最大间隔，使分割落在自然停顿处。
- **空白字幕显示** —— 相邻字幕间隔超过 0.1 秒时自动填充空白项，用 `. . .` 点占位符直观呈现时长，时间轴连续完整。
- **分割视频** —— 长视频可按 1 小时或 30 分钟分割，分成较短的段落。
- **导出音频** —— 视频可以导出原声音频或者适合 whisper 的压缩音频，方便外部使用。

### 体验优化

- **字幕渲染性能大幅提升** —— 全面优化长字幕的渲染策略，视口外段落简化渲染、减少 DOM 节点开销。此前 3~5 小时字幕已出现明显卡顿，现在即使处理 10+ 小时超长字幕仍流畅丝滑。
- **文稿匹配算法改善** —— 重构匹配逻辑，增加多余处理和边界情况修复，匹配准确度进一步提升。
- **Python 环境检测优化** —— 启动时优先检测系统 Python，若已安装 whisper 和 scenedetect 则直接使用，跳过内嵌运行时解压，首次启动速度大幅提升。
- **右键菜单图标统一** —— 素材和时间线的右键菜单图标风格一致化，视觉更协调。

## 功能总览

### 素材与字幕

| 功能                | 说明                                                             |
| ------------------- | ---------------------------------------------------------------- |
| 多格式导入          | 支持 MP4、WebM、MOV、MP3、WAV、JPG、PNG 等主流格式               |
| 导入外部字幕        | SRT、VTT、ASS/SSA、SUB、LRC 等格式，字幕文件可拖动排序                                      |
| 目录导入            | 选择目录，自动扫描并按名称前缀匹配视频与字幕                     |
| 生成空白字幕        | 按智能时长（约 5 分钟）自动分段，在自然停顿处切割                |
| Whisper AI 语音识别 | 内嵌 Python + Whisper，自动生成带时间戳字幕；优先使用系统 Python |
| 智能场景分割        | 使用 scenedetect 自动检测场景切换点                              |
| 字幕复制            | 右键快速复制已有字幕                                             |
| VTT 字幕导出        | 将编辑后的字幕导出为 VTT 文件                                    |

### 字幕编辑

| 操作         | 说明                                                 |
| ------------ | ---------------------------------------------------- |
| 鼠标悬停     | 实时预览对应视频帧和音频                             |
| 点击         | 选中/取消选中字幕条目                                |
| 按住拖动     | 批量选中字幕                                         |
| Shift + 拖动 | 批量取消选中                                         |
| B 键         | 分割字幕（基于时间比例）                             |
| D 键         | 合并分隔线两侧的字幕                                 |
| 空格键       | 播放/暂停当前视频                                    |
| P 键         | 截取当前帧到桌面                                     |
| 页面搜索     | 搜索字幕文字，支持 `?` `*` 模糊匹配，高亮 + 导航 |

### 时间线与导出

| 功能          | 说明                                          |
| ------------- | --------------------------------------------- |
| 多时间线      | 创建多条独立时间线，互不干扰，支持拖拽排序    |
| 时间线复制    | 快速复制已有时间线                            |
| 文稿匹配      | 粘贴文稿自动匹配字幕并生成时间线（专业版）    |
| 视频导出      | H.264 编码，CRF 18 高质量，支持自定义输出目录 |
| FCPXMLD 导出  | 与 Final Cut Pro，达芬奇，剪映等无缝衔接（专业版）           |
| FCP7 XML 导出 | 与 Adobe Premiere Pro 无缝衔接（专业版）      |
