## 什么是 X-Cut？

**X-Cut** 是一款基于 Electron 开发的跨平台桌面视频剪辑工具，支持 macOS 和 Windows。它的核心理念只有六个字：**用字幕，剪视频**。

与传统剪辑软件不同，X-Cut 没有复杂的多轨时间轴，没有眼花缭乱的特效转场。它把字幕作为剪辑的核心——你看到字幕文字，就能选中对应的视频片段；你编辑字幕，就是在编辑视频。

这是一种更直觉、更高效的剪辑方式，特别适合播客剪辑、访谈精剪、教程整理等场景。

## 为什么选择 X-Cut？

### 1. 字幕驱动，所见即所得

传统剪辑软件的操作逻辑是：在时间轴上拖拽素材、标记入点出点、切割合并。而 X-Cut 的操作逻辑是：**读文字，选片段**。

导入视频后，X-Cut 可以：

- 自动识别语音生成字幕（内置 OpenAI Whisper）
- 导入 SRT、VTT、ASS 等外部字幕
- 按场景智能分割视频片段

然后，你只需要在字幕列表里浏览文字内容，点击或拖拽选中想要的片段，就能完成剪辑。鼠标悬停在字幕上，还能实时预览对应的视频画面和音频波形。

### 2. 极简交互，零学习成本

X-Cut 的界面采用经典三栏布局：左侧素材面板、中间字幕编辑器、右侧时间线和预览。所有操作都围绕字幕展开：

| 操作         | 功能                |
| ------------ | ------------------- |
| 点击字幕     | 选中/取消选中片段   |
| 按住拖动     | 批量选中多个片段    |
| Shift + 拖动 | 批量取消选中        |
| B 键         | 在光标处分割字幕    |
| D 键         | 合并相邻字幕        |
| 空格键       | 播放/暂停           |
| I 键         | 编辑字幕文字        |
| U 键         | AI 重新识别当前字幕 |

没有复杂的快捷键记忆负担，几分钟就能上手。

### 3. AI 加持，自动化处理

X-Cut 内嵌了完整的 Python 3.12 运行环境和 FFmpeg，首次启动自动配置，无需手动安装：

- **Whisper 语音识别**：自动为视频生成高精度字幕
- **智能场景分割**：自动检测视频场景切换点，生成分段字幕
- **逐字时间稿**：输出精确的单词级时间戳，支持去除特定词汇（如"嗯""啊"等口癖）

### 4. 专业衔接，不锁数据

剪辑完成后，X-Cut 支持多种导出方式：

- **视频导出**：H.264 编码，CRF 18 高质量，自动统一分辨率
- **VTT 字幕导出**：保留编辑后的字幕文件
- **FCPXMLD 导出**：直接导入 Final Cut Pro、DaVinci Resolve、剪映专业版
- **FCP7 XML 导出**：与 Adobe Premiere Pro 无缝衔接

你可以把 X-Cut 作为粗剪工具，快速筛选素材后，再导入专业软件进行精剪、调色、加特效。
