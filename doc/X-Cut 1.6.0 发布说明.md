## v1.6.0 带来了 AI 转写能力，批量去掉特定词等新功能

### 新功能

- **去除特定词** —— 基于逐字时间稿，智能去除字幕中的特定词（如口语赘词「啊」「呢」「那个」等），自动切分字幕并取消选中被移除的词，一键清理音频内容。
- **逐字时间稿生成** —— Whisper 生成字幕时同时输出 JSON 格式的逐字时间稿（包含每个词的起止时间），为精确的字幕编辑提供数据基础，文字更干净、分离音乐等效果更好。
- **AI 转写服务** —— 内置 Whisper HTTP 服务架构，支持对指定时间段音频进行转写；鼠标悬停字幕条目按 **U** 键可重新识别该段文字内容。
- **字幕跟随播放** —— 播放视频时，字幕区自动高亮当前播放位置对应的字幕条目并滚动到可视区域，实时追踪播放进度。
- **段落导航与播放** —— 左侧字幕点击显示段落列表（含起止时间），点击段落可快速跳转或从该位置开始播放。
- **目录递归导入** —— 目录导入支持递归扫描子文件夹，自动按名称前缀匹配深层目录中的视频与字幕文件。
- **JSON 字幕支持** —— 字幕文件新增 JSON 格式支持，可直接导入 Whisper 输出的 JSON 字幕文件。
- **字幕导出文本** —— 字幕文件支持导出为纯文本格式，方便在本地编辑器中进行批量处理。

### 体验优化

- **Whisper 模型升级** —— 默认模型从 `turbo` 切换为 `small`，转录速度提升约一倍，内存占用降至约 2GB（原 turbo 约 6GB）。
- **交互模式统一** —— 素材/字幕的右键菜单、时间线右键菜单、段落扩展操作全部改为图标点击触发，视觉更一致。
- **预览性能优化** —— 多项性能调优：播放器内联、时间线切换波形图闪动修复、空闲检测间隔调整，降低 CPU 占用。
- **打包问题修复** —— 解决了长期存在的应用打包后资源文件找不到的问题，确保发布版本稳定可用。
- **依赖配置增强** —— 依赖安装路径支持绝对路径配置，适应更多部署场景。
- **UI 细节改善** —— 字幕边框样式优化便于区分、工具按钮添加提示文案、段落标题显示结束时间、素材点击即打开播放器。

### 问题修复

- 修复转录服务超时导致长时间空跑的问题
- 修复切换时间线时字幕误跳转的问题
- 修复字幕文本编辑保存异常的问题
- 修复场景分割后字幕更新不完整的问题

## 功能总览

### 素材与字幕

| 功能                | 说明                                                         |
| ------------------- | ------------------------------------------------------------ |
| 多格式导入          | 支持 MP4、WebM、MOV、MP3、WAV、JPG、PNG 等主流格式           |
| 导入外部字幕        | SRT、VTT、ASS/SSA、SUB、LRC、JSON 等格式，字幕文件可拖动排序 |
| 目录导入            | 选择目录，递归扫描子文件夹，自动按名称前缀匹配视频与字幕     |
| 生成空白字幕        | 按智能时长（约 5 分钟）自动分段，在自然停顿处切割            |
| Whisper AI 语音识别 | 内嵌 Python + Whisper，默认 small 模型，自动生成带时间戳字幕 |
| 逐字时间稿          | 生成字幕同时输出 JSON 逐词时间戳，支持去除特定词等精确操作   |
| 智能场景分割        | 使用 scenedetect 自动检测场景切换点                          |
| 字幕文本编辑        | 双击直接编辑字幕文字内容                                     |
| AI 转写刷新         | 悬停字幕按 U 键调用 Whisper 重新识别该段文字                 |
| 去除特定词          | 基于逐字时间稿智能去除口语赘词等特定词汇                     |
| 字幕复制            | 右键快速复制已有字幕                                         |
| 字幕导出文本        | 导出为纯文本文件，方便本地批量编辑                           |
| VTT 字幕导出        | 将编辑后的字幕导出为 VTT 文件                                |

### 字幕编辑

| 操作         | 说明                                                 |
| ------------ | ---------------------------------------------------- |
| 鼠标悬停     | 实时预览对应视频帧和音频                             |
| 点击         | 选中/取消选中字幕条目                                |
| 按住拖动     | 批量选中字幕                                         |
| Shift + 拖动 | 批量取消选中                                         |
| B 键         | 分割字幕（基于时间比例）                             |
| D 键         | 合并分隔线两侧的字幕                                 |
| 空格键       | 播放/暂停当前视频                                    |
| P 键         | 截取当前帧到桌面                                     |
| I 键         | 编辑当前字幕条目                                     |
| U 键         | AI 重新识别当前字幕文字                              |
| S 键         | 切割当前段落（合并段落在段落标题栏扩展菜单）         |
| 段落点击     | 跳转到该段落或从该位置开始播放                       |
| 页面搜索     | 搜索字幕文字，支持 `?` `*` 模糊匹配，高亮 + 导航 |

### 时间线与导出

| 功能          | 说明                                               |
| ------------- | -------------------------------------------------- |
| 多时间线      | 创建多条独立时间线，互不干扰，支持拖拽排序         |
| 时间线复制    | 快速复制已有时间线                                 |
| 文稿匹配      | 粘贴文稿自动匹配字幕并生成时间线（专业版）         |
| 视频导出      | H.264 编码，CRF 18 高质量，支持自定义输出目录      |
| 视频分割      | 长视频按 1 小时或 30 分钟分割                      |
| 音频导出      | 导出原声音频或压缩音频（适合 whisper 使用）        |
| FCPXMLD 导出  | 与 Final Cut Pro，达芬奇，剪映等无缝衔接（专业版） |
| FCP7 XML 导出 | 与 Adobe Premiere Pro 无缝衔接（专业版）           |
