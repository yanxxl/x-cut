## v1.5.0 带来字幕编辑能力和段落管理功能

### 新功能

- **字幕文本编辑** —— 字幕条目支持 I 键进入编辑模式，可修改字幕文字内容，按 Enter 或点击保存，ESC 取消编辑。输入法使用更流畅，不影响其他快捷键操作。
- **字幕段落管理** —— 字幕可按约 3 分钟（可自定义时长）分段为多个段落，每个段落显示标题栏（包含段号、起止时间和字幕数量），长字幕浏览更清晰。
- **段落分割与合并** —— 支持将段落从任意位置分割成两段，或向上合并相邻段落，灵活调整分段结构。
- **字幕批量选择** —— 新增全选/全不选按钮，配合 Shift+拖动批量取消选择，操作更高效。

### 体验优化

- **依赖管理重构** —— 全新设计的依赖安装流程，界面更紧凑直观，支持 Whisper 和 scenedetect 的独立一键安装。
- **问题修复** —— 修复上个版本视频导出时找不到 ffmpeg 的问题。

## 功能总览

### 素材与字幕

| 功能                | 说明                                                             |
| ------------------- | ---------------------------------------------------------------- |
| 多格式导入          | 支持 MP4、WebM、MOV、MP3、WAV、JPG、PNG 等主流格式               |
| 导入外部字幕        | SRT、VTT、ASS/SSA、SUB、LRC 等格式，字幕文件可拖动排序           |
| 目录导入            | 选择目录，自动扫描并按名称前缀匹配视频与字幕                     |
| 生成空白字幕        | 按智能时长（约 5 分钟）自动分段，在自然停顿处切割                |
| Whisper AI 语音识别 | 内嵌 Python + Whisper，自动生成带时间戳字幕；优先使用系统 Python |
| 智能场景分割        | 使用 scenedetect 自动检测场景切换点                              |
| 字幕文本编辑        | 双击直接编辑字幕文字内容                                         |
| 字幕复制            | 右键快速复制已有字幕                                             |
| VTT 字幕导出        | 将编辑后的字幕导出为 VTT 文件                                    |

### 字幕编辑

| 操作         | 说明                                                 |
| ------------ | ---------------------------------------------------- |
| 鼠标悬停     | 实时预览对应视频帧和音频                             |
| 点击         | 选中/取消选中字幕条目                                |
| 双击         | 进入字幕文本编辑模式                                 |
| 按住拖动     | 批量选中字幕                                         |
| Shift + 拖动 | 批量取消选中                                         |
| B 键         | 分割字幕（基于时间比例）                             |
| D 键         | 合并分隔线两侧的字幕                                 |
| 空格键       | 播放/暂停当前视频                                    |
| P 键         | 截取当前帧到桌面                                     |
| I 键         | 编辑当前字幕条目                                     |
| S 键         | 切割当前段落（合并段落在段落标题栏扩展菜单）         |
| 页面搜索     | 搜索字幕文字，支持 `?` `*` 模糊匹配，高亮 + 导航 |

### 时间线与导出

| 功能          | 说明                                               |
| ------------- | -------------------------------------------------- |
| 多时间线      | 创建多条独立时间线，互不干扰，支持拖拽排序         |
| 时间线复制    | 快速复制已有时间线                                 |
| 文稿匹配      | 粘贴文稿自动匹配字幕并生成时间线（专业版）         |
| 视频导出      | H.264 编码，CRF 18 高质量，支持自定义输出目录      |
| 视频分割      | 长视频按 1 小时或 30 分钟分割                      |
| 音频导出      | 导出原声音频或压缩音频（适合 whisper 使用）        |
| FCPXMLD 导出  | 与 Final Cut Pro，达芬奇，剪映等无缝衔接（专业版） |
| FCP7 XML 导出 | 与 Adobe Premiere Pro 无缝衔接（专业版）           |
