## v1.3.0 带来多项重要新功能和体验优化

### 新功能

- **文稿匹配** —— 粘贴一段文稿，系统自动模糊匹配对应字幕片段，并一键生成时间线。基于 fuse.js 模糊搜索 + LCS 二次匹配算法，支持相邻字幕合并搜索，匹配精准高效。
- **字幕复制** —— 右键菜单新增"复制一份"选项，可快速复制已有字幕，方便安排选中字幕的顺序。
- **时间线复制** —— 右键菜单新增"复制一份"选项，可复制现有时间线，在已有方案基础上快速迭代。
- **VTT 字幕导出** —— 将生成的字幕导出为 VTT 文件到桌面，方便在其他工具中使用。
- **视频帧截图** —— 按下 P 键，截取当前预览帧到桌面，支持视频和图片素材。
- **页面内容搜索** —— 在字幕编辑器中搜索文字内容，支持高亮显示、上下导航、ESC 关闭。
- **LRC 字幕格式支持** —— 导入字幕新增 LRC 格式。

### 体验优化

- **视频导出增加目录选择** —— 导出时可自定义输出目录，不再固定为桌面。
- **编辑器按钮布局优化** —— 改善顶部工具栏按钮位置，操作更顺手。
- **窗口大小自动记忆** —— 关闭后再次打开，自动恢复上次的窗口大小。

## 功能总览

### 素材与字幕

| 功能 | 说明 |
|------|------|
| 多格式导入 | 支持 MP4、WebM、MOV、MP3、WAV、JPG、PNG 等主流格式 |
| 导入外部字幕 | SRT、VTT、ASS/SSA、SUB、LRC |
| 生成空白字幕 | 按固定时长（25秒）自动分段 |
| Whisper AI 语音识别 | 内嵌 Python + Whisper，自动生成带时间戳字幕 |
| 智能场景分割 | 使用 scenedetect 自动检测场景切换点 |
| 字幕复制 | 右键快速复制已有字幕 |
| VTT 字幕导出 | 将编辑后的字幕导出为 VTT 文件 |

### 字幕编辑

| 操作 | 说明 |
|------|------|
| 鼠标悬停 | 实时预览对应视频帧和音频 |
| 点击 | 选中/取消选中字幕条目 |
| 按住拖动 | 批量选中字幕 |
| Shift + 拖动 | 批量取消选中 |
| B 键 | 分割字幕（基于时间比例） |
| D 键 | 合并分隔线两侧的字幕 |
| 空格键 | 播放/暂停当前视频 |
| P 键 | 截取当前帧到桌面 |
| 页面搜索 | 搜索字幕文字，高亮 + 导航 |

### 时间线与导出

| 功能 | 说明 |
|------|------|
| 多时间线 | 创建多条独立时间线，互不干扰 |
| 时间线复制 | 快速复制已有时间线 |
| 文稿匹配 | 粘贴文稿自动匹配字幕并生成时间线（专业版） |
| 视频导出 | H.264 编码，CRF 18 高质量，支持自定义输出目录 |
| FCPXMLD 导出 | 与 Final Cut Pro 无缝衔接（专业版） |
