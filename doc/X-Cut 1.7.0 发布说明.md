## v1.7.0 带来拖放导入、软件信息面板和多项体验优化

### 新功能

- **拖放导入** —— 首页支持直接拖放视频文件和文件夹导入素材，自动匹配字幕，操作更直观高效。
- **软件信息面板** —— 首页新增应用信息区域，展示软件名称、版本号、功能说明和依赖状态，一目了然。
- **专业版激活状态** —— 独立的激活状态组件，实时显示专业版状态（正常/试用中/已过期），过期前自动提醒剩余天数。
- **素材文件重新选择** —— 素材右键菜单新增「重新选择文件」，可更换素材源文件而保留已有字幕数据。

### 体验优化

- **首页重构** —— 简化列表结构，拆分为独立组件，改善说明文案，整体布局更清晰。
- **预览播放增强** —— 预览区显示当前播放的素材名称及时长；片段播放更完整流畅；文件不存在时给出明确提示。
- **字幕段落交互改善** —— 播放图标支持 hover 高亮和点击反馈；段落状态与点击导航联动，操作更流畅。
- **字幕显示效果改善** —— 优化字幕条目的视觉呈现，更易阅读和区分。
- **字幕文件删除确认** —— 删除字幕文件前弹出确认对话框，防止误操作。
- **下拉菜单交互修复** —— 下拉菜单点击不再冒泡触发上级元素事件，避免误触。
- **依赖检测策略优化** —— Whisper 和 scenedetect 改为目录存在即判定已安装，避免系统依赖缺失导致版本检查失败；异常捕获并输出日志，排查更方便。
- **场景检测缓存版本号** —— 版本号缓存后无需重复检测，提升启动速度。
- **Whisper 命令参数优化** —— 添加参数避免错误提示和繁体中文输出问题。
- **文件夹导入增强** —— 从文件夹导入素材时，同步处理 JSON 逐字时间稿文件。
- **视频导出改善** —— 导出视频时，片段独立放置，目录清晰不杂乱。

### 问题修复

- 修复时间线数据丢失的问题
- 修复字幕分段偶现空段的问题
- 修复逐字时间稿单词丢失空格的问题

## 功能总览

### 素材与字幕

| 功能                | 说明                                                               |
| ------------------- | ------------------------------------------------------------------ |
| 多格式导入          | 支持 MP4、WebM、MOV、MP3、WAV、JPG、PNG 等主流格式                 |
| 拖放导入            | 拖放文件或文件夹即可导入视频和字幕，并自动匹配字幕，操作更直观高效 |
| 导入外部字幕        | SRT、VTT、ASS/SSA、SUB、LRC、JSON 等格式，字幕文件可拖动排序       |
| 目录导入            | 选择目录，递归扫描子文件夹，自动按名称前缀匹配视频与字幕           |
| 生成空白字幕        | 按智能时长（约 5 分钟）自动分段，在自然停顿处切割                  |
| Whisper AI 语音识别 | 内嵌 Python + Whisper，默认 small 模型，自动生成带时间戳字幕       |
| 逐字时间稿          | 生成字幕同时输出 JSON 逐词时间戳，支持去除特定词等精确操作         |
| 智能场景分割        | 使用 scenedetect 自动检测场景切换点                                |
| 字幕文本编辑        | 双击直接编辑字幕文字内容                                           |
| AI 转写刷新         | 悬停字幕按 U 键调用 Whisper 重新识别该段文字                       |
| 去除特定词          | 基于逐字时间稿智能去除口语赘词等特定词汇                           |
| 字幕文件复制        | 右键快速复制已有字幕                                               |
| 字幕导出文本        | 导出为纯文本文件，方便本地批量编辑                                 |
| VTT 字幕导出        | 将编辑后的字幕导出为 VTT 文件                                      |

### 字幕编辑

| 操作         | 说明                                                 |
| ------------ | ---------------------------------------------------- |
| 鼠标悬停     | 实时预览对应视频帧和音频                             |
| 点击         | 选中/取消选中字幕条目                                |
| 按住拖动     | 批量选中字幕                                         |
| Shift + 拖动 | 批量取消选中                                         |
| B 键         | 分割字幕（基于时间比例）                             |
| D 键         | 合并分隔线两侧的字幕                                 |
| 空格键       | 播放/暂停当前视频                                    |
| P 键         | 截取当前帧到桌面                                     |
| I 键         | 编辑当前字幕条目                                     |
| U 键         | AI 重新识别当前字幕文字                              |
| S 键         | 切割当前段落（合并段落在段落标题栏扩展菜单）         |
| 段落点击     | 跳转到该段落或从该位置开始播放                       |
| 页面搜索     | 搜索字幕文字，支持 `?` `*` 模糊匹配，高亮 + 导航 |

### 时间线与导出

| 功能          | 说明                                               |
| ------------- | -------------------------------------------------- |
| 多时间线      | 创建多条独立时间线，互不干扰，支持拖拽排序         |
| 时间线复制    | 快速复制已有时间线                                 |
| 文稿匹配      | 粘贴文稿自动匹配字幕并生成时间线（专业版）         |
| 视频导出      | H.264 编码，CRF 18 高质量，支持自定义输出目录      |
| 视频分割      | 长视频按 1 小时或 30 分钟分割                      |
| 音频导出      | 导出原声音频或压缩音频（适合 whisper 使用）        |
| FCPXMLD 导出  | 与 Final Cut Pro，达芬奇，剪映等无缝衔接（专业版） |
| FCP7 XML 导出 | 与 Adobe Premiere Pro 无缝衔接（专业版）           |
