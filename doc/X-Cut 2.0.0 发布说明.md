## X-Cut 是什么

X-Cut 是一款基于字幕的视频剪辑工具。

它的核心理念很简单：**用字幕，剪视频**。

传统的视频剪辑，你需要在时间轴上反复拖拽、切割、对齐。而在 X-Cut 中，你只需要阅读字幕——选中你想保留的句子，删除不需要的段落，然后导出。剪辑过程就像编辑一篇文档。

### 为什么能用字幕来剪视频

这要从视频文件的内部结构说起。

视频文件由**容器**（Container）包裹，里面封装着多条**轨道**（Track）：视频轨道、音频轨道，以及可选的字幕轨道。这些轨道通过**时间戳**（Timestamp）保持同步——每一帧画面、每一段声音、每一句字幕，都知道自己该在什么时刻出现。

字幕文件（如 SRT、VTT）本质上就是一份**时间戳对照表**：

```
1
00:00:01,000 --> 00:00:04,500
大家好，欢迎来到本期节目

2
00:00:04,800 --> 00:00:08,200
今天我们要聊一个有趣的话题
```

这意味着：字幕天然就是视频的**索引**。当你选中一句字幕，你实际上选中的是从 `00:00:01.000` 到 `00:00:04.500` 的这一段视频和音频。

X-Cut 做的事很简单：**根据字幕的时间戳，对视频进行无损切割和重组**。

你不需要手动标记入点出点，不需要在波形上寻找气口，不需要逐帧对齐——字幕已经帮你做好了这一切。你只需要阅读文字，决定哪些内容值得保留。

### 为什么用字幕剪辑更自然高效

视频的本质是信息载体。而字幕，是信息的最小单元。

当你用字幕剪辑时：

- 你看到的是**内容**，而非波形
- 你操作的是**语义**，而非时间码
- 你思考的是**故事**，而非技术参数

对于访谈、播客、课程、Vlog 这类以语言为主要信息载体的视频，字幕剪辑的效率是传统方式的 5-10 倍。

### 使用方法

X-Cut 的工作流分为三步：

**1. 导入与识别**

- 直接拖入视频文件或文件夹，自动导入并匹配字幕
- 或点击按钮，手动选择要导入的内容
- 使用内置 Whisper AI 自动生成字幕
- 或导入已有的 SRT/VTT/ASS 字幕文件

**2. 选择与删除**

- 浏览字幕列表，像阅读文稿一样审阅内容
- 选中需要保留的片段（或删除不需要的）
- 支持批量选择、反向选择、段落操作

**3. 导出与衔接**

- 一键导出剪辑后的视频
- 或导出 FCPXML/FCP7 XML，在传统剪辑软件中继续精修

整个过程，你几乎不需要接触时间轴。

X-Cut 不是万能剪辑工具，但它在以下场景表现卓越：

| 场景                    | 说明                                 |
| ----------------------- | ------------------------------------ |
| **播客/访谈剪辑** | 去除口癖、停顿、重复内容，快速出片   |
| **课程/教程制作** | 删除录制失误，保留核心知识点         |

如果你经常需要处理「说话的视频」，X-Cut 会是你工作流中不可或缺的一环。

### 素材与字幕

| 功能         | 说明                                          |
| ------------ | --------------------------------------------- |
| 多格式导入   | MP4、WebM、MOV、MP3、WAV、JPG、PNG 等主流格式 |

| 字幕导出     | 导出为 VTT 或纯文本文件                       |

| 音频导出      | 导出原声或压缩音频                     |

| 操作         | 说明                                   |

| 鼠标悬停     | 实时预览对应视频帧和音频               |

| U 键         | AI 重新识别当前字幕文字                |
