# X-Cut 1.8.1 发布说明

## 新增功能

### 导出选项：合并相邻条目

在导出设置中新增了「合并相邻条目」选项。

**用途**：
- 当字幕条目中存在时间相邻的片段时，可选择是否将其合并为单一视频片段
- 合并后导出的视频片段更少，导出更快
- 不合并则保留原始分割逻辑，适合需要精细控制每个片段的场景
